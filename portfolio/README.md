# Portfolio

Portfolio personal de Victor Puerta construido con Astro.

## Stack

- Astro 6
- CSS local en componentes y una capa global minima en el layout
- Node `>=22.12.0`

## Desarrollo

```bash
nvm use
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notas

- El proyecto ya no usa Tailwind ni `@astrojs/tailwind`.
- La guia visual viva esta en [`DESIGN.md`](./DESIGN.md).
- La version esperada de Node tambien esta fijada en [`.nvmrc`](./.nvmrc).
